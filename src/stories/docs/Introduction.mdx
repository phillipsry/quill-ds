## Quick start

```bash
# Clone
git clone <repo>
cd quill-ds

# Install
npm install

# Dev
npm run dev          # Next.js at localhost:3000
npm run storybook    # Storybook at localhost:6006
```

## Foundation

| Layer | Tool |
|---|---|
| Framework | Next.js 16, App Router |
| Styling | Tailwind CSS v4, CSS custom properties |
| Components | ShadCN (source-owned), Base UI primitives |
| Fonts | Fraunces (display, variable), Raleway (UI) |
| Icons | Material Symbols Outlined (custom `<Icon>`) |

See `DESIGN.md` in the repo for the full token specification.
