# Spacing

Quill uses a 4px base grid. All spacing tokens are multiples of 4px.

## Usage guide

| Scale | Token | Use |
|---|---|---|
| 4–8px | `space-1`, `space-2` | Icon gaps, badge insets, hairline offsets |
| 12–16px | `space-3`, `space-4` | Component internals, input padding, small card spacing |
| 20–24px | `space-5`, `space-6` | Form spacing, card padding, list row gaps |
| 32–48px | `space-8`, `space-12` | Section padding, modal insets, comfortable breathing room |
| 64–96px | `space-16`, `space-24` | Page-level padding, large section separators |
