## Fraunces presets

Fraunces is a variable font with optical-size (`opsz`), softness (`SOFT`), and wonkiness (`WONK`) axes.

## Eyebrow pattern

Used for section labels, category tags, and UI micro-copy.

<div style={{ padding: '16px 0' }}>
  <div style={{ display: 'flex', alignItems: 'center', gap: 8 }}>
    <div style={{ width: 18, height: 2, background: '#C4684B', borderRadius: 1 }}/>
    <span style={{ fontFamily: 'var(--font-inter, Inter, sans-serif)', fontSize: 11, fontWeight: 500, textTransform: 'uppercase', letterSpacing: '0.15em', color: '#675F58' }}>Category · Eyebrow Label</span>
  </div>
</div>

```css
.eyebrow {
  font-size: 11px;
  font-weight: 500;
  text-transform: uppercase;
  letter-spacing: 0.15em;
  color: var(--ink-muted);
}
/* Preceded by a 2px terracotta dash via ::before or a sibling element */
```
